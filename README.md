# theurge test project

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

## Tech Stack

In the project, React, TypeScript, Storybook, TailwindCSS was used.

## Run project

Install dependencies using `yarn install` command.
To run the project, run `yarn start` command in the project folder.
To run the storybook, run `yarn storybook` command in the project folder.
